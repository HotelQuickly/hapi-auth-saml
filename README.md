# hapi-auth-saml

Hapi.js scheme for SAML authentication

## Development

Install pre-requisites

    npm install

Run `gulp` for linting

    gulp

## Examples

Install pre-requisites

    npm install

Create a `examples/secrets.js` (see code for what is required).

Run example

    node examples/ping-federate.js

Or run with npm

    npm run ping
